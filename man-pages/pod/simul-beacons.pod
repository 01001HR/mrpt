=head1 NAME

simul-beacons - Robot dataset simulator

=head1 SYNOPSIS

simul-beacons I<config_file.ini>

=head1 DESCRIPTION

B<simul-beacons> is a command-line application which generates a synthetic rawlog of a simulated robot 
as it follows a predefined path and observes a set of beacons (through range-only measurements).

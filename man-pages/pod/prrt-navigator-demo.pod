=head1 NAME

prrt-navigator-demo - Mobile robot navigation simulator 

=head1 SYNOPSIS

prrt-navigator-demo

=head1 DESCRIPTION

B<prrt-navigator-demo> is a GUI program that allows experimenting with a 
new robot navigation algorithm based on Rapidly-exploring Random Tree (RTT) 
exploration using Parameterized Trajectory Generators (PTGs). 

Refer to the source code and related papers for more information about the 
method itself.

=head1 NAME

kf-slam - SLAM method using Kalman filtering

=head1 SYNOPSIS

kf-slam I<config_file.ini>

=head1 DESCRIPTION

B<kf-slam> is a command-line application which processes a rawlog and 
create a map of landmarks using a 6D SLAM algorithm.

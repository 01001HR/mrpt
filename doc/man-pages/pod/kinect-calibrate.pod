=head1 NAME

kinect-calibrate - Calibration utility for the Microsoft Kinect sensor

=head1 SYNOPSIS

kinect-calibrate

=head1 DESCRIPTION

B<kinect-calibrate> is designed to allow an easy calibration of the 
camera intrinsic and extrinsic parameters required for an accurate
reconstruction of colored 3D point clouds with the sensor. 
Calibration will require a printed checkerboard as those used
in common camera calibration.
